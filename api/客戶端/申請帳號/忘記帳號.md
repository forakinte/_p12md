# CPKG-120000-NotificationSystem

## 客戶端 > 忘記帳號 API

> 編輯(程序)

| 項目                      | 內容                       | 說明                |驗證                      |   備註         |
|---------------------------|----------------------------|----------------------|-----------------|----------------|
| <b>路徑</b>               | notificationSystem/client-account/forget    |                        |                |                  |
| <b>方法</b>               | PUT                        |                    |                    |                 |
| <b>權限</b>               | -                       |                     |                   |                 |
| <b>參數</b>               |                            |                       |                 |                 |
|                           | email       | 電子信箱            | 必填,字串,需符合email格式         |                 |
